# Requirements

- Git setup and credentials
- Ansible setup
- `ansible-galaxy install -r requirements.yml -p ./roles`

# Run

```bash
ansible-playbook mac-setup.yml --ask-become-pass
```

# Update the list

```bash
npm list -g --depth=0
pip list
```
